# AGENT.md

## Project Overview

Build a WordPress plugin named **Euphoria**.

The plugin should create a React-powered WordPress admin page that replicates the provided Figma design for:

**WooCommerce Review Widget → Email Reminder**

Only this screen needs to be implemented.

---

## Goal

Create a production-quality WordPress plugin demonstrating:

* WordPress Plugin Architecture
* React Admin Application
* REST API Integration
* Settings Persistence
* Reusable Components
* Modern Development Practices

### Composer

PHP classes in `includes/` are autoloaded via Composer classmap.

```bash
composer install
```

`euphoria.php` loads `vendor/autoload.php` when present.

## Scope

Implement only:

* Sidebar navigation (static)
* Settings tab
* First Email tab (static)
* Follow-ups tab (static)
* Email Reminder Toggle
* General Settings section
* Timing Settings section
* Delivery Settings section
* Ratings Scale section
* Redirection section
* Email Preview panel
* Desktop / Mobile preview switch

Do NOT implement:

* WooCommerce integration
* Email sending
* Analytics
* Automation workflows
* Dynamic navigation

---

## Data Persistence

All settings must:

* Save to WordPress database
* Load automatically on page refresh

Recommended:

```php
update_option( 'euphoria_email_reminder_settings', $data );
get_option( 'euphoria_email_reminder_settings', [] );
```

---

## REST API

Create endpoints:

### GET

```http
/wp-json/euphoria/v1/settings
```

Returns saved settings.

### POST

```http
/wp-json/euphoria/v1/settings
```

Saves settings.

Requirements:

* nonce verification
* capability check
* sanitization
* validation

---

## Security Requirements

Always:

* Sanitize incoming data
* Escape output
* Verify permissions
* Verify nonce

Use:

```php
sanitize_text_field()
sanitize_email()
absint()
wp_verify_nonce()
current_user_can()
```

Never trust client-side data.
